# my-pnpm

Quickly enter commands for pnpm


# Installation
```cmd
npm install -g my-pnpm
```

# Usage

## n

```js
n 
// pnpm install

n -h 
// pnpm -h
```

## nd 

```js
nd webpack
// pnpm i -D webpack

nd eslint typescript
// pnpm i -D eslint typescript
```

## nt

```js
nt react
// pnpm i -D @types/react

nt lodash-es node
// pnpm i -D @types/lodash-es @types/node

```

## nx

```js
nx tsc -v
// pnpm exec tsc -v
```

## other
```
use `n`
```
